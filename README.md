# Blog Preview Card
A simple and elegant Blog Preview Card built with HTML and CSS. It highlights a post with its title, publish date, summary, and author details, while including hover and focus states for better interactivity.

## Overview
A simple solution to the [Frontend Mentor Blog Preview Card challenge](https://www.frontendmentor.io/challenges/blog-preview-card-ckPaj01IcS).  
The goal was to build a responsive blog preview card that matches the provided design as closely as possible using only HTML and CSS.

---

## Features
- Responsive design for mobile and desktop
- Semantic HTML structure
- Hover and focus states for interactive elements
- Clean, minimal CSS styling

---

## Technologies
- HTML5  
- CSS3 (Flexbox / Grid)
